# Welcome to LSC SCAN
LSC Scan is a static security analysis tool (Static Analysis Security Testing â€“ SAST) designed specifically to analyze Laravel 12 codebases. It acts as an automated audit assistant that scans source files locally to identify risky coding patterns that may lead to security vulnerabilities. The main goal of this tool is to detect common issues such as SQL Injection, Cross-Site Scripting (XSS), and environment configuration mistakes that often occur during the application development lifecycle.

# How to run in your environment?
```bash
git clone https://github.com/Yu1chiro/sc-security-scan.git
pip install -r requirements.txt
python main.py
```
